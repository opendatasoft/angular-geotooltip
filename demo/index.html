Angular-GeoTooltip live examples

Hover the underlined text to display the tooltip!

> More info on the GitHub repository

2D tooltip

A simple tooltip to display a point with default settings. 

<geotooltip coords="pointCoordinates">My decorated text</geotooltip>

2D tooltip with specific sizes

Display a point with a specific size defined in the directive's attributes. 

<geotooltip coords="pointCoordinates" width="400" height="300">My decorated text</geotooltip>

2D tooltip with instant pop

Simple tooltip to display a point, but with no delay to pop. 

<geotooltip coords="pointCoordinates" delay="0">My decorated text</geotooltip>

GeoJSON tooltip

A simple tooltip with default settings, but this time to display a GeoJSON shape. 

<geotooltip geojson="geoJson">My decorated text</geotooltip>

GeoJSON + 2D large tooltip

A tooltip with a larger size, containing both a GeoJSON shape and a point. 

<geotooltip geojson="geoJson" coords="pointCoordinates" delay="0" width="600" height="400">My decorated text</geotooltip>
